Краткое руководство пользователей сервиса TaskVizor

Доступ к веб-сервису

Сервис доступен по адресу https://app.taskvizor.by

Авторизация в веб-приложение

Для начала работы необходимо войти в приложение, используя предоставленные логин и пароль, и выбрав язык из списка:

Создание сотрудника

Для создания сотрудника необходимо перейти в справочники, нажав на шестеренку в правом верхнем углу и выбрав соответствующий пункт из выпадающего списка:

В открывшейся вкладке нажать кнопку добавления нового сотрудника:

Откроется окно с полями для создания нового сотрудника. Заполнить необходимые поля (помечены *) и нажать кнопку Добавить:

Сотрудник будет добавлен и отобразиться в таблице.

Создание точки

Для создания точки необходимо перейти в таблицу точек, нажав на шестеренку в правом верхнем углу и выбрав соответствующий пункт из выпадающего списка:

В открывшейся вкладке нажать кнопку добавления новой точки:

Откроется окно с полями для создания новой точки и картой. Найти на карте необходимое здание/адрес и нажать на него правой кнопкой мыши. В форме создания точки появится строка с выбранным адресом, на который необходимо нажать для автоматического заполнения адресной строки:

Заполнить оставшиеся обязательные поля (помечены *). Для того, чтобы точка была доступна сотруднику, необходимо её привязать. Выбрать из списка сотрудников, к которым необходимо привязать точку. Сохранить созданную точку.

Создание шаблона задания

Для создания шаблона задания необходимо перейти в справочники, нажав на шестеренку в правом верхнем углу и выбрав соответствующий пункт из выпадающего списка:

Перейти во вкладку Шаблоны заданий. Нажать кнопку добавления нового шаблона:

Откроется форма для создания нового шаблона задания. Для начала, необходимо добавить новый вид работ, к которому будет относиться форма. Для этого необходимо нажать на плюс рядом с полем выбора вида, заполнить обязательные поля, и нажать кнопку Добавить:

Заполнить обязательные поля, выбрать иконку и добавить новую форму. Форма создана, в выпадающих списках отображается вид и тип созданной формы. В блоке слева выбрать элемент, который необходимо добавить в форму, и перетянуть в центральное окно. При перетаскивании элемента появится красный прямоугольник, в который и разместится элемент.

После добавления элемента, справа появится окно свойств, которые необходимо заполнить. Для каждого элемента доступность полей может варьировать. После добавления элементов, нажать кнопку Сохранить:

Создание маршрута

Во вкладке Планирование нажать кнопку для создания нового маршрута:

В открывшемся окне выбрать сотрудника для назначения маршрута, даты начала и окончания. Также есть возможность задать правило повторения маршрута, отметив флаг Повторяющийся маршрут и указав дни повторения.

Сохранить созданный маршрут нажатием кнопки ОК.

Создание задания

Во вкладке Планирование выбрать точку в блоке слева и перетащить в созданный маршрут. Если точка не привязана к сотруднику, появится диалог для привязки точки к сотруднику. Разрешить привязку точки.

Нажать иконку редактирования добавленной точки в маршруте:

В открывшемся окне для редактирования заполнить обязательные поля. По умолчанию маршрут создается на весь день, чтобы указать временной интервал необходимо убрать флаг «На весь день». Появятся поля для указания времени и даты начала и окончания задания. Дата окончания задания не может быть позднее даты окончания маршрута.

Сохранить внесенные изменения. Задача поставлена и отображается в маршруте с обновленным статусом.

Формирование отчетов

Формирование базового отчета

Перейти во вкладку Отчеты. В выпадающем списке выбрать базовый отчет:

Выбрать диапазон дат для формирования отчета в открывшемся календарном окне. Выбрать одного или несколько сотрудников из списка:

Нажать кнопку Выполнить. Отчет появится в блоке под картой:

Отчет сгруппирован по сотруднику и по дате, для просмотра подробной информации необходимо развернуть отчет нажатием иконки .

Для формирования отчета за прошедшую неделю или за вчера можно воспользоваться быстрыми кнопками, при нажатии на которые будет сформирован отчет:

Для сохранения отчета в xlsx-формате нажать кнопку .

Формирование сводного отчета

Перейти во вкладку Отчеты. В выпадающем списке выбрать сводный отчет:

Выбрать диапазон дат для формирования отчета в открывшемся календарном окне. Выбрать одного или несколько сотрудников из списка:

Нажать кнопку Выполнить. Отчет появится в блоке под картой:

Отчет сгруппирован по сотруднику и по дате, для просмотра подробной информации необходимо развернуть отчет нажатием иконки .

Для формирования отчета за прошедшую неделю или за вчера можно воспользоваться быстрыми кнопками, при нажатии на которые будет сформирован отчет:

Для сохранения отчета в xlsx-формате нажать кнопку .

Авторизация в мобильное приложение

После того, как задача поставлена, необходимо авторизоваться в приложении, используя данные сотрудника, на которого была назначена задача, и выбрав сервер, соответствующий вашему региону:

Просмотр задач

Для просмотра поставленных задач перейти в Задачи в главном окне приложения.

Откроется список задач на текущий день. Вы можете выбрать любой период для отображения, переходя по вкладкам День, Неделя, Месяц и Период:

Нажать на задачу. Откроется окно с основной информацией по задаче, картой и данными заказчика.

При нажатии на номер телефона, будет запущено приложения для осуществления вызова. Нажатие на маркер на карте вызовет диалог со списком навигаторов для построения оптимального маршрута:

Выполнение задачи

Для того, чтобы приступить к выполнению задачи, необходимо нажать кнопку Начать. Откроется окно с таймером, элементами для заполнения и кнопками завершения и отмены.

Отмена задачи

Для отмены задачи необходимо нажать кнопку Отменить. В открывшемся окне необходимо выбрать причину отмены из выпадающего списка, и по возможности прикрепить фото и подпись:

Завершение задачи

Для завершения задачи необходимо заполнить все обязательные элементы формы, и нажать кнопку Закончить, которая станет активной после заполнения всех полей:

После завершения задачи, её статус в общем списке изменится: зеленый - задача завершена, красный - отменена, серый - не синхронизирована с сервером:

Уведомления

При постановке новых задач или изменении текущих, на мобильное устройство приходят push-уведомления:

Для просмотра подробной информации, нажать на уведомление. Откроется окно чата:

После получения уведомлений рекомендуется обновить список задач. Для этого необходимо дождаться автоматического обновления, либо воспользоваться кнопкой ручной синхронизации:

Подтверждение задачи*

[* - данные функции можно включить/отключить в настройках веб-приложения]

Для подтверждения задачи необходимо нажать на иконку статуса задачи, либо нажать на саму задачу в списке.

После этого задача будет считаться подтвержденной и статус задачи изменится.

Приостановка выполнения задачи*

Чтобы приостановить выполнение задачи, необходимо нажать на кнопку паузы в окне выполнения задачи и подтвердить приостановку в появившемся диалоге.

Задача приостановлена. Статус задачи обновлен:

Возобновление выполнения задачи*

Чтобы возобновить выполнение задачи необходимо нажать на статус задачи. Откроется окно выполнения задачи с запущенным таймером и с информацией, которая была заполнена до приостановки:

Также можно нажать на задачу в общем списке, затем нажать кнопку Продолжить:

Выполнение задачи также будет возобновлено.
